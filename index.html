How to use

Each line of the above text area represents either 'metadata' or a 'command'.

The first line you prefix with '#' becomes the title of your new workout.

The second line you prefix with '#' becomes the expanded description of your new workout.

Any following lines that you prefix with '#' will get ignored by the parser.

Available Commands:

beep => beeps.

sleep X => Waits X seconds before executing the next command.

sleepOffset X O => Waits between X-O and X+O seconds before executing the next command.

say X => Says 'X' via the text-to-speech engine.

Flow Control:

Blocks => Wrapping commands with {...}N makes these commands a block which will be executed N times
